# Expense Tracker Web App

A web-based Expense Tracker built using Python.

## 🔧 Features
- Add, edit, and delete income
- Track expenses with categories
- Automatic savings calculation
- Dashboard with reports
- User-friendly interface

## 🛠️ Technologies Used
- Python
- Flask 
- HTML
- CSS
