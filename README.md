# Expense Tracker Web App

A web-based Expense Tracker built using Python.

## 🔧 Features
- Add, edit, and delete income
- Track expenses with categories
- Automatic savings calculation
- Dashboard with reports
- User-friendly interface

## 🛠️ Technologies Used
- Python
- Flask (if used)
- HTML
- CSS

## 🚀 How to Run
1. Install Python
2. Install required packages:
   pip install flask
3. Run the app:
   python app.py
4. Open browser:
   http://127.0.0.1:5000/
